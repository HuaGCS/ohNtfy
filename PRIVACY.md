# 隐私政策 / Privacy Policy

**更新日期 / Last Updated**: 2026-06-17

---

## 中文

### 数据收集说明
本应用（NtfyPush）非常重视您的隐私。我们承诺不会收集、存储或上传任何您的个人身份信息。

### 我们收集的数据
本应用本身不收集任何个人数据。应用仅将以下数据存储在您的设备本地：

- 订阅的主题列表（服务器地址、主题名称、显示名称）
- 最近接收的消息缓存
- 应用设置（默认服务器、轮询间隔、通知偏好等）

以上所有数据仅保存在设备本地，不会传输到任何第三方服务器。

### 第三方服务
本应用是一个 ntfy 推送服务的客户端。您连接的 ntfy 服务器由您自行选择（默认为 ntfy.sh）。

当您使用 ntfy 服务时，您的消息会经过您指定的 ntfy 服务器中转。请您自行了解所使用的 ntfy 服务器的隐私政策。

ntfy.sh 的隐私政策请参考：https://ntfy.sh/privacy

### 网络权限
本应用需要网络访问权限以连接您指定的 ntfy 服务器，实现消息的收发功能。网络请求仅在您订阅的主题范围内进行，不会用于其他用途。

### 通知权限
本应用需要通知权限以在收到新消息时向您推送通知。您可以随时在系统设置中关闭通知权限。

### 后台运行
本应用需要后台运行权限以确保能够持续接收推送消息。后台运行时，应用仅保持必要的网络连接以接收消息更新，不会执行任何额外的数据收集操作。

### 数据删除
您可以随时通过以下方式删除本地数据：

- 删除单个订阅：在首页长按订阅 → 删除
- 清除所有数据：在系统设置中清除应用数据
- 卸载应用：将删除所有本地存储的数据

### 政策更新
本隐私政策可能会不时更新。如有重大变更，我们将在应用内通知您。继续使用本应用即表示您同意本隐私政策的条款。

### 联系方式
如果您对本隐私政策有任何疑问或建议，请通过 Gitea 仓库提交 Issue：
https://gitea.huagcs.com/OpenHarmony/ntfy/issues

## English

### Data Collection Statement
NtfyPush values your privacy. We do not collect, store, or upload any personally identifiable information.

### Data We Collect
This application itself does not collect any personal data. The following data is stored **locally on your device only**:

- Subscribed topic list (server URL, topic name, display name)
- Recent message cache
- App settings (default server, polling interval, notification preferences, etc.)

All data above is stored locally on your device and is **never transmitted** to any third-party server.

### Third-Party Services
This app is a client for the ntfy push notification service. The ntfy server you connect to is chosen by you (default: ntfy.sh).

When you use the ntfy service, your messages will be relayed through the ntfy server you specify. Please review the privacy policy of your chosen ntfy server.

ntfy.sh privacy policy: https://ntfy.sh/privacy

### Network Permission
This app requires network access to connect to your specified ntfy server for sending and receiving messages. Network requests are limited to your subscribed topics only.

### Notification Permission
This app requires notification permission to display push notifications when new messages arrive. You can disable this at any time in system settings.

### Background Operation
This app requires background operation permission to continuously receive push messages. While running in the background, the app only maintains necessary network connections for message updates and does not perform any additional data collection.

### Data Deletion
You can delete local data at any time:

- Delete a single subscription: Long press subscription on home screen → Delete
- Clear all data: Clear app data in system settings
- Uninstall the app: Removes all locally stored data

### Policy Updates
This privacy policy may be updated from time to time. For material changes, we will notify you within the app. Continued use of this app constitutes acceptance of this policy.

### Contact
If you have any questions or suggestions regarding this privacy policy, please submit an Issue on the Gitea repository:
